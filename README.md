# Avengers Universe Blog 🦸‍♂️

Blog moderno sobre el universo de los Avengers (Marvel) con diseño cinematográfico.

## 🎨 Características

- **Diseño Cinematográfico**: Inspirado en el estilo visual de Marvel con colores oscuros y acentos vibrantes
- **Responsive**: Optimizado para dispositivos móviles, tablets y escritorio
- **React Router**: Navegación fluida entre páginas
- **4 Artículos**: Análisis, teorías, rankings y noticias sobre Avengers
- **Sistema de Comentarios**: UI preparada para integración con Disqus
- **Componentes Reutilizables**: Arquitectura modular y escalable

## 🎯 Estructura

```
/src
  /app
    /components    - Componentes reutilizables
    /pages        - Páginas de la aplicación
    /data         - Datos de los posts
    routes.ts     - Configuración de rutas
    App.tsx       - Componente principal
  /styles         - Estilos globales y fuentes
```

## 🚀 Despliegue en Netlify

1. Conecta tu repositorio a Netlify
2. Configura el build:
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
3. Netlify detectará automáticamente Vite
4. Despliega
